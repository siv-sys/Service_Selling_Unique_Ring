# Backend Scripts

Use this folder for runnable one-off backend tasks.

```text
scripts/
  database/      Database bootstrap and schema setup
  utilities/     Diagnostics, test helpers, and maintenance scripts
  seedRings.js   Local catalog seed data
```

Common commands from `backend/`:

```bash
npm run db:init
npm run db:seed
npm run users:seed
npm run test:api
npm run test:connection
```
